# vite-plugin-agent-skills

Bun monorepo for [`vite-plugin-agent-skills`](packages/vite-plugin-agent-skills),
a Vite plugin for importing Markdown and Agent Skills from static import
attributes.

- Package docs, usage, and config: [packages/vite-plugin-agent-skills/README.md](packages/vite-plugin-agent-skills/README.md)
- Example app: [examples/basic](examples/basic)

## Development

```sh
bun install
bun run build
bun run typecheck
bun run test
bun run lint
bun run format:check
```
